# 🛣️ Roadmap

A practical phased roadmap for evolving EUT-Core into a robust research platform.

## Phase 1 — Structure
- Finalize repository organization and naming conventions.
- Establish canonical docs pages and cross-links.
- Define minimal contribution and citation workflow.

## Phase 2 — Formalization
- Tighten axiom statements and notation consistency.
- Expand tensor and geometry derivation notes in `theory/`.
- Record assumptions, limits, and recovery conditions explicitly.

## Phase 3 — Demonstrations
- Implement reproducible notebook demos for WE1, WE3, and WE5.
- Add shared analysis scripts in `code/analysis/`.
- Publish example datasets and baseline visualizations.

## Phase 4 — Publication Integration
- Align manuscript folders with publication-ready structure.
- Connect releases to Zenodo snapshots and DOIs.
- Maintain changelogs linking theory updates to manuscript revisions.

## Phase 5 — Broader Ecosystem
- Support external contributors and replication studies.
- Add interoperability hooks for adjacent tooling.
- Curate educational and onboarding assets for new collaborators.
