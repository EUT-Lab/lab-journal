# 🧭 Repository Map

## Top-level directories

- `docs/` — Documentation landing pages, conceptual summaries, maps, and roadmap.
- `theory/` — Formal framework development.
  - `theory/axioms/` — Axiom refinements and formal variants.
  - `theory/geometry/` — Emergent geometry notes and derivations.
  - `theory/harmonic-field/` — Harmonic field definitions and dynamics.
  - `theory/tensor-formulation/` — Tensor-level formulations and calculations.
  - `theory/recovery-limits/` — Recovery conditions and effective-limit analyses.
- `papers/` — Manuscript workspaces.
  - `papers/eut-theory/`
  - `papers/eut-cosmology/`
  - `papers/eut-gravitational-waves/`
  - `papers/eut-biology/`
- `notebooks/` — Reproducible computational notebooks.
  - `notebooks/lensing/`
  - `notebooks/cosmology/`
  - `notebooks/gw-polarization/`
  - `notebooks/harmonic-networks/`
- `code/` — Shared code modules.
  - `code/simulations/`
  - `code/analysis/`
  - `code/utilities/`
- `figures/` — Visual assets.
  - `figures/diagrams/`
  - `figures/plots/`
  - `figures/visualizations/`
- `datasets/` — Data assets.
  - `datasets/example-data/`
- `references/` — Supporting literature and notes.
