If you are new to EUT, this order is recommended:

2. [Framework Map](framework-map.md) — how key components connect.
3. [Axioms](axioms.md) — minimal assumptions and formal stance.
4. [Working Elements](working-elements.md) — modular research programs WE1–WE8.
