# 🌌 EUT-Core

**The canonical repository for the Extended Unified Theory framework**

EUT-Core is the central research hub for the Extended Unified Theory (EUT): an open, evolving framework that explores how physical structure can emerge through **resonance before form**. This repository is organized as a lab-grade home for theory development, computational demonstrations, manuscripts, references, and reproducible research assets.

## 🚀 Start Here

New to the project? Begin with the documentation landing page:

- [`docs/index.md`](docs/index.md)

## 🧭 Repository Structure

- `docs/` — Human-readable research documentation, onboarding, roadmap, and framework guides.
- `theory/` — Formal theory scaffolds, axioms, geometry, tensor formulations, and limits.
- `papers/` — Manuscript workspaces organized by major EUT domains.
- `notebooks/` — Computational notebooks for demonstrations and exploratory analysis.
- `code/` — Reusable simulation, analysis, and utility modules.
- `figures/` — Diagrams, plots, and visual outputs for papers and docs.
- `datasets/` — Example and benchmark data assets.
- `references/` — Literature notes, bibliographic material, and paper references.

For a full directory-by-directory map, see [`docs/repo-map.md`](docs/repo-map.md).

## 🧠 Core Concepts

- **Harmonic field `H`** as the primitive organizing substrate.
- **Geometry map `Φ(H) → (g, ∇)`** as the bridge from harmonic structure to emergent geometric description.
- **Working elements WE1–WE8** as modular research programs connecting formal assumptions to testable consequences.
